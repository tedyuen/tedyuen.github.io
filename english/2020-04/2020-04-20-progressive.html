# Progressive -- 渐进式

BrE /prə'ɡrɛsɪv/ ; AmE /prə'ɡrɛsɪv/

  1. adj. 不断前进的, 有进展的, 逐渐上升的
  2. 进步的, 先进的

副词: progressively 名词: progressiveness

# 释义

  1. Progressive framework

让我们先来看一段文字：

Vue (pronounced /vjuː/, like view) is a progressive framework for building user interfaces. Unlike other monolithic frameworks, Vue is designed from the ground up to be incrementally adoptable.

Vue (读音 /vjuː/，类似于 view) 是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue被设计为可以自底向上逐层应用。

以上是Vue官方文档里对这个框架的介绍，称之为渐进式框架。首先我们先来理解什么是渐进式框架。

知乎上有一个被尤大点赞的解释：主张最少，没有强主张。即给了我们一套完整的解决方案，但不给我们提什么要求。我们用了他的视图模板引擎，可以不用他的路由，可以不用他的状态管理，甚至可以和jQuery一起用。

  1. Progressive Web App

百度百科: PWA（Progressive Web App）是一种理念，使用多种技术来增强web app的功能，可以让网站的体验变得更好，能够模拟一些原生功能，比如通知推送。在移动端利用标准化框架，让网页应用呈现和原生应用相似的体验。

2015年Google提出了渐进式网页应用，定义为渐进式接近原生App的Web App。当用户从手机主屏幕启动时，不用考虑网络的状态是如何，都可以立刻加载出PWA。可以不用从应用市场下载而直接通过网站把入口添加到主屏(Android)。

# 最后

从以上两段例子看过来，我们可以把这个单词理解成慢慢来，不用全部都学完，也不用全部都必须用。比如你学了九阳神功第一重你就可以出去选武林盟主了，你可以考虑回来继续学后面几重，但是不影响你第一重就可以出去拉风。渐进式不但包括使用的渐进式，也包括开发的渐进式，好的设计模式支持无限的扩展，有无限的可能。

当小伙伴们想要开发一款这样的工具或框架，就可以用这个单词来描述他哦。
