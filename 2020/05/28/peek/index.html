Peek -- 偷看

Ted Yuen   in Shanghai
Thu May 28 2020
  • Vocabulary

BrE /piːk/ ; AmE /pik/

  1. n. 偷看；一瞥，看一眼
  2. vi. 窥视，偷看

# 栗子

This extension extends Vue code editing with Go To Definition and Peek Definition support for components and filenames in single-file components with a .vue extension. It allows quickly jumping to or peeking into files that are referenced as components (from template), or as module imports (from script). 当你编写在一个扩展名为.vue的一个单文件组件中编写vue的代码，此扩展支持直接跳转和瞥一眼组件或文件名的定义位置。它也允许你快速的跳转或瞥一眼文件模板中的组件或以模块形式引入的定义位置。
