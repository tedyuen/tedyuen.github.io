Tutorial -- 使用说明书

Ted Yuen   in Shanghai
Wed Apr 22 2020
  • Vocabulary

BrE /tjuː'tɔːrɪəl/ ; AmE /tu'tɔrɪəl/

  1. n. (大学导师的) 个别辅导时间，辅导课;
  2. 教程; 辅导材料; 使用说明书;
  3. adj. 导师的; 私人教师的; 辅导的;

# 栗子

# 1. Tutorial / Guide / Manual

分开看都认识，放一起是不是忽然发现不知道啥时候用什么了？

  • Guide: 用户指南 提供一些简要信息的时候。
  • Manual: 用户手册 提供完整的深入说明，和Guide差不多，没有标准说多少字用Guide，多少字用Manual。
  • Tutorial: 使用说明书、教程 当需要给没用经验的人一步步的指导时使用，我们在Github上见到这个词，说明作者要开始扫盲了。

# 2. xxx tutorial

  • ES6 tutorial -- 不多说，阮一峰的开源ES6教程，懂的入。
  • Python tutorial -- 一位国内大神整理的一篮子关于机器学习的手册。
  • Machine learning tutorials -- 另一个机器学习的高星项目。
  • TensorFlow tutorials -- Hvass Labs的TensorFlow教程

# 最后

作为名词时，没什么其他的，注意拼写和发音，别当自己在写README的时候，一时想不起来怎么拼写。个人理解为苦口婆心的手把手教授的意境。
